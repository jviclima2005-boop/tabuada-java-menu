# 📊 Gerador de Tabuada em Java

Aplicação desenvolvida em Java com menu interativo no terminal, permitindo ao usuário gerar a tabuada de qualquer número.

Este projeto foi criado com o objetivo de praticar lógica de programação e reforçar conceitos fundamentais da linguagem Java.

---

## 🚀 Demonstração

```bash
=== MENU ===
1 - Calcular Tabuada
0 - Sair
Escolha: 1

Digite o número: 5

Tabuada do 5:
5 x 1 = 5
5 x 2 = 10
...
```

---

## 🎯 Objetivo

Desenvolver habilidades essenciais em programação, como:

* Controle de fluxo
* Interação com o usuário via terminal
* Organização de código
* Estruturação de um programa simples e funcional

---

## 🧠 Conceitos aplicados

* Estruturas condicionais (`if`, `else if`, `else`)
* Laço de repetição (`while`, `for`)
* Entrada de dados com `Scanner`
* Boas práticas (fechamento de recursos com `close()`)

---

## ⚙️ Tecnologias utilizadas

* Java
* JDK (Java Development Kit)

## ▶️ Como executar o projeto

1. Clone o repositório:

```bash
git clone https://github.com/seu-usuario/tabuada-java-menu.git
```

2. Acesse a pasta:

```bash
cd tabuada-java-menu
```

3. Compile o código:

```bash
javac Ex03.java
```

4. Execute o programa:

```bash
java Ex03
```
