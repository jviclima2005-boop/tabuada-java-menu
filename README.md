# 📊 Gerador de Tabuada em Java

Este projeto foi desenvolvido como parte dos meus estudos em lógica de programação utilizando Java.

A aplicação apresenta um menu interativo no terminal, onde o usuário pode escolher gerar a tabuada de um número ou encerrar o programa.

---

## 🎯 Funcionalidades

- Menu interativo com opções
- Geração de tabuada de 1 a 10
- Estruturas de repetição (`while` e `for`)
- Entrada de dados com `Scanner`

---

## 🧠 Conceitos praticados

- Estrutura de decisão (`if/else`)
- Laços de repetição
- Entrada e saída de dados
- Organização de fluxo com menu
